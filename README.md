## 사용


- `react` 사용자 인터페이스 구축
- `framer-motion` 애니메이션
- `react-query` 비동기 데이터 요청 및 데이터 상태 관리
- `react-hook-form` 폼 관리 및 유효성 검사
- `react-icons` 아이콘 사용
- `typescript` 정적 타입 적용
